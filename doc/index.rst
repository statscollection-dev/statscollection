Statscollection
===============

.. https://dbader.org/blog/write-a-great-readme-for-your-github-project

Purpose and scope
-----------------

Statscollection is a collection of packages for statistical analysis and modeling written in Python.
The purpose of Statscollection is to act as a repository of statistical software, where contributions and users can gather.

.. note::
   The package is under development, so APIs might change without warning.
   See the respective sub-packages for more information.



Installation
-----------------

Use `pip <https://pypi.org/project/statscollection/>`_ to install Statscollection.

.. code-block:: console

   $ pip install statscollection
   
If you do not have Python installed, the `Anaconda distribution <https://www.anaconda.com/distribution/>`_ is recommended.
